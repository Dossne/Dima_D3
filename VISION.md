# VISION.md — Tap Miner: Collapse Run
*Единый источник правды: что мы делаем, зачем и как это ощущается*

---

## Одна фраза

> Игрок ведёт майнера вниз по вертикальной шахте, тапая влево/вправо между тремя лейнами. Сверху давит обвал, снизу — лут и опасности. Каждый ран — 20–50 секунд. Смерть → апгрейд → снова.

---

## Референсы

| Игра | Что берём |
|------|-----------|
| **Tomb of the Mask** | Вертикальный скролл, portrait, читаемые паттерны, аркадный темп, мгновенный рестарт |
| **Archero** | Upgrade-петля между ранами, idle-lite прогрессия, ощущение роста |
| **Downwell** | Спуск вниз, нарастающая угроза, roguelite ощущение, короткие раны |
| **Crossy Road / Flappy Bird** | Простое управление, честная смерть, моментальный рестарт |

**Формула:** `Tomb of the Mask readability + mining fantasy + Archero-style meta-lite`

│ [CollapseBar]  [42.5m]  │  ← HUD: полоска обвала + глубина

│                         │
│   ▓▓▓▓▓ COLLAPSE ▓▓▓▓▓  │  ← обвал надвигается сверху
│                         │
│  [ H ]  [   ]  [ H ]   │
│  [   ]  [ R ]  [   ]   │  ← сегмент шахты с блоками
│  [   ]  [   ]  [ S ]   │
│  [ S ]  [ S ]  [ S ]   │
│                         │
│        [🧑‍⚒️]             │  ← персонаж на 35% от низа

- Шахта **скроллится снизу вверх** — персонаж визуально "спускается"
- Камера держит персонажа на **35% от нижнего края** экрана
- Первые **5–8 секунд** — максимально простые сегменты, туториал не нужен

---

## Лейны и движение персонажа

**3 фиксированных лейна** (Left / Center / Right). Лейны не меняют положение, не двигаются горизонтально.

```
Lane 0     Lane 1     Lane 2
(Left)    (Center)   (Right)
  X=-w      X=0        X=+w
```

**Управление:**
- Tap left → переход в соседний левый лейн
- Tap right → переход в соседний правый лейн
- Движение **дискретное** — всегда на якорь лейна, никаких промежуточных позиций
- Время перехода: **120 мс** (анимация наклона + движение)
- В буфер принимается **1 следующий input** во время перехода
- Auto-correction **запрещён** — куда тапнул, туда и пошёл

## Что происходит на экране во время рана

### 1. Сегменты шахты
Поле разбито на сегменты глубиной 5–12 рядов. Каждый ряд — 3 ячейки. Содержимое:

| Символ | Что это |
|--------|---------|
| **S** | Safe — пустая проходимая ячейка |
| **R** | Reward — блок с лутом, ломается автоматически при входе |
| **H** | Hazard — опасность, телеграфируется за 0.5 сек до удара |
| **_** | Пусто — нет объектов |

**Блоки ломаются автоматически** — персонаж входит в ячейку → auto-break → лут → дальше. Отдельной атаки нет.

### 2. Hazard'ы
Бьют по HP если персонаж в том же лейне в момент активации. **Всегда телеграфируются за 0.5 сек** (цвет, анимация, звук) — смерть от hazard'а должна быть объяснима.

Типы (MVP): шипы, взрывные блоки, блоки-ловушки, ядовитый газ.

### 3. CollapseWall — обвал сверху
Надвигается сверху вниз с нарастающей скоростью:

| Глубина | Скорость |
|---------|---------|
| Depth 0 (старт) | 1.8 units/sec |
| Depth 10 | 2.4 units/sec |
| Depth 25+ | 3.2 units/sec |

Догнал игрока → **мгновенная смерть** (cause = COLLAPSE).  
За 0.6 сек до контакта — visual + audio telegraph (shake, цвет, звук).

### 4. Лут и монеты
Подбираются автоматически при прохождении через ячейку с Reward. CoinLabel обновляется мгновенно. Редкий лут (кристаллы) виден заранее — провоцирует риск.

### 5. HUD (всегда на экране)
- **CollapseBar** — fill-полоска, показывает близость обвала
- **DepthLabel** — текущая глубина, обновляется каждый кадр
- **CoinLabel** — монеты текущего рана

## Паттерны сегментов — примеры

### Early (Depth 0–10) — читаемость важнее давления
```
Lane0  Lane1  Lane2
  S      R      S    ← reward в центре (небольшой риск)
  S      S      S
  H      S      S    ← hazard снизу, telegraph виден
  S      S      S
```
Всегда есть очевидный safe path. Обучаем: reward = риск.

### Mid (Depth 11–25) — давление нарастает
```
Lane0  Lane1  Lane2
  R      H      R    ← два пути с наградой, оба требуют уклонения
  S      _      S
  H      S      H    ← центр = fallback
  S      S      S
```
Safe path не всегда самый выгодный. Collapse давит. Нужно принимать решения быстро.

### Late (Depth 25+) — skill check
```
Lane0  Lane1  Lane2
  R      H      R
  H      S      H
  S      R      S
  H      H      S    ← единственный путь: lane2 → lane1 → lane2
```
Тесные маршруты, быстрые решения, высокая цена ошибки.

### Запрещённые паттерны
- `H H H` в одном ряду — unfair death, нарушает Fairness Contract
- Reward без safe path рядом — ловушка, не выбор
- Hazard без telegraph — смерть необъяснима
- 3+ ряда hazard подряд — читаемость рушится

## Враги — есть ли они?

На MVP — **нет полноценных врагов**. Есть **enemy-hazards** — hazard-объекты с базовым поведением (не боёвка):

- Стационарные hazard-существа (как статичные шипы, но анимированные)
- Движущиеся моб-типы (крот, слизень) — появятся в VS3

Враги — это **живые препятствия**, а не combat targets. Игрок не атакует врагов, он их **обходит**.

## Как меняется сложность и визуал по глубине

| Глубина | Геймплей | Цвет среды |
|---------|----------|------------|
| **0–10 Early** | Простые паттерны, очевидный safe path, медленный collapse | Серый камень `#48484A`, тёплые коричневые акценты |
| **11–25 Mid** | Safe path не всегда выгоден, сложные паттерны | Тёмный камень `#2C2C2E` + синие кристаллы `#00C7FF` |
| **25+ Late** | Быстрые решения, тесные маршруты, collapse давит | Почти чёрный `#1C1C1E` + purple neon glow `#BF5AF2` |

## Петля смерти и рестарта

```
Ран (20–50 сек)
      ↓
    Смерть
      ↓
 Results Screen
 (глубина + монеты + breakdown)
      ↓
 Upgrade Screen
 (купить 1–2 апгрейда)
      ↓
 Новый ран — быстро
```

**Главный принцип:** от смерти до следующего тапа "старт" — **не более 5–10 секунд**. Смерть не раздражает, она **запускает следующий цикл**.

---

## Pillars — чем руководствуемся при каждом решении

| Пиллар | Что это значит на практике |
|--------|---------------------------|
| **Мгновенно понятно** | Управление и цель ясны за 5 секунд без туториала |
| **Плотный темп** | Игрок почти всегда принимает микро-решение |
| **Риск против награды** | Глубже = опаснее = выгоднее. Жадность — движущая сила |
| **Быстрый рестарт** | Смерть не штрафует, а перезапускает |
| **Лёгкая мета** | Прогресс ощущается уже после 2–3 забегов |

---

## Fairness Contract — что никогда не нарушаем

- **Readable route** — всегда есть хотя бы один понятный маршрут
- **Fair death** — тестер должен объяснить причину смерти без помощи разработчика
- **Telegraph** — hazard виден до контакта, когда требуется навык
- **Pressure** — collapse наказывает за стояние, не за нормальную игру
- **Restart** — смерть быстро переходит в следующую попытку
